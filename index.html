Usage Examples

Basic Usage

package main

import (
  "fmt"
  "github.com/bedirhangull/protolex"
)

func main() {
  // Read and parse proto file
  proto, err := protolex.ReadProtoFile("./path/to/your.proto")
  if err != nil {
    fmt.Printf("Error reading proto file: %v\n", err)
    return
  }

  // Get all message blocks
  messages := proto.GetAllMessageBlocks()
  for _, msg := range messages {
    fmt.Println("Message:", msg)
  }

  // Get all enum blocks 
  enums := proto.GetAllEnumBlocks()
  for _, enum := range enums {
    fmt.Println("Enum:", enum)
  }
}

API Reference

Comprehensive documentation of all available methods in Protolex, organized by functionality. Click a category to jump to its details.

Message Operations Enum Operations Service Operations Package Operations Other Operations

Message Operations

Functions for working with Protocol Buffer messages

GetAllMessageBlocks() Returns []string

Returns all message blocks in the proto file

GetMessageByName(name string) Returns string

Returns a specific message block by its name

GetAllFieldsByMessageName(messageName string) Returns []string, error

Returns all fields defined in a message

GetAllTypesByMessageName(messageName string) Returns []string, error

Returns all types used in a message

Enum Operations

Functions for working with Protocol Buffer enums

GetAllEnumBlocks() Returns []string

Returns all enum blocks in the proto file

GetEnumByName(name string) Returns string

Returns a specific enum block by its name

GetAllEnumFieldsByEnumName(enumName string) Returns []string, error

Returns all fields defined in an enum

GetAllEnumsByMessageName(messageName string) Returns []string, error

Returns all enums used in a message

Service Operations

Functions for working with Protocol Buffer services

GetAllServiceBlocks() Returns []string

Returns all service blocks in the proto file

GetServiceByName(name string) Returns string

Returns a specific service block by its name

GetRPCByServiceName(serviceName string, RPCname string) Returns string

Returns a specific RPC definition from a service

Package Operations

Functions for working with Protocol Buffer packages

GetAllPackages() Returns []*Package, error

Returns all packages defined in the proto file

GetPackageByPath(path string) Returns *Package, error

Returns a specific package by its full path

GetChildPackagesByPath(parentPath string) Returns []*Package, error

Returns all child packages of a given package

GetParentPackageByPath(childPath string) Returns *Package, error

Returns the parent package of a given package

Other Operations

Additional utility functions for Protocol Buffer files

GetSyntax() Returns string

Returns the syntax declaration of the proto file

GetPackageName() Returns string

Returns the package declaration of the proto file

Protolex is an open-source Go package for parsing and analyzing Protocol Buffer files. It provides a comprehensive set of tools to extract and analyze various components of proto files.
